## 프로젝트 소개 📝

인공지능을 활용하여 안심하고 공유할 수 있는 카셰어링 서비스


차량 대여자와 사용자는 인공지능을 활용하여 차량의 파손 유무를 미리 확인하여 대여 및 반납할 수 있습니다!
